## Description
The Status component is used to indicate the status of a service, API call, health check, etc.

## Usage and Style guidelines

For usage and style guidelines please refer to [status indicators](/patterns/status-indicators/style)

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the styles:

    ```js
    @import '@console/pal/Components/Status/styles';
    ```

3.  Import the component:

    ```js
    import { Status } from '@console/pal/Components';
    ```

4.  Render the component.

For general component usage review the [Components Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/components/overview).
