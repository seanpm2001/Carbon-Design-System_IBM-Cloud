## Description
Translate is a component used for embedding HTML or React components inside a translated string. It serves the same purpose as the react-i18next Trans component but behaves a bit differently. You wrap the relevant text inside the string with a tag and then provide the properties to apply to the element / component created by that tag. Three tags are currently supported: `a`, `code`, and `span`. Use the `a` tag to embed links in the text, `code` to embed inline code, and `span` to apply any classes or styling to certain words.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the styles:

    ```js
    @import '@console/pal/Components/Translate/styles';
    ```

3.  Import the component:

    ```js
    import { Translate } from '@console/pal/Components';
    ```

4.  Render the component.

For general component usage review the [Components Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/components/overview).
