## Description
The Solution tile can be used to refer to offerings/solutions on cloud or display other content.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the styles:

    ```js
    @import '@carbon/ibm-cloud/components/SolutionTile/styles';
    ```

3.  Import the component:

    ```js
    import { SolutionTile, SolutionTileHeader, SolutionTileDescription, SolutionTileDetails, SolutionTileTags, SolutionTileIcon } from '@carbon/ibm-cloud/components';
    ```

4.  Render the component.

For general component usage review the [Components Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/components/overview).
