## Description

The TagList component is used to render a list of tags. It is used primarily for user defined (resource) tags. The TagList is primarily used in the connected TagList, TagArea, and TagModal components. The TagList can be seen on a resource details page where tags are associated to the resources

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the styles:

    ```js
    @import '@console/pal/Components/TagList/styles';
    ```

3.  Import the component:

    ```js
    import { TagList } from "@console/pal/Components";
    ```

4.  Render the component.

For general component usage review the [Components Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/components/overview).
