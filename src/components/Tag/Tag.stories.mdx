## Description

The Tag component is a wrapper around Carbon's tag component. It is used for rendering user defined (resource) Tags on the cloud platform. The blue tags are reserved for user defined Tags and should not be used for any other form of tagging.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the styles:

    ```js
    @import '@console/pal/Components/Tag/styles';
    ```

3.  Import the component:

    ```js
    import { Tag } from "@console/pal/Components";
    ```

4.  Render the component.

For general component usage review the [Components Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/components/overview).
