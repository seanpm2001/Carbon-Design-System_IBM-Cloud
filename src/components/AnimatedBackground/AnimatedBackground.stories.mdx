## Description

AnimatedBackground is a component used as a background of world level pages in Cloud PAL experimental style. It can be rendered with or without motion and offers two color-sets.

All properties to control the animation are exposed to consumers, but it is advised to only alter them if needed.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the styles:

    ```js
    @import '@console/pal/Components/AnimatedBackground/styles';
    ```

3.  Import the component:

    ```js
    import { AnimatedBackground } from "@console/pal/Components";
    ```

4.  Render the component.

For general component usage review the [Components Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/components/overview).
