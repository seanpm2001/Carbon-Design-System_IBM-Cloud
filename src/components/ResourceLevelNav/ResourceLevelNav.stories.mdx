## Description
A component used for the navigation of pages within a resource on a resource details page. It should be used as outlined in the Resource Page Template. This component DOES NOT have nested elements
the convention is to use Level 4 (L4) Horizontal tabs instead. [Here](/ibm-cloud-concepts/navigation-on-cloud/usage#navigation-levels) a good resource on using the horizontal navigation on the cloud.
Skeleton states on the resource level nav do not display below 672px as it transforms into a collapsible drawer on small view-ports.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the styles:

    ```js
    @import '@console/pal/Components/ResourceLevelNav/styles';
    ```

3.  Import the component:

    ```js
    import { ResourceLevelNav } from '@console/pal/Components';
    ```

4.  Render the component.

For general component usage review the [Components Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/components/overview).
