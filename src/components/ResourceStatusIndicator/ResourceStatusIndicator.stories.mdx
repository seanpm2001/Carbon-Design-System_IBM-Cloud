## Description

ResourceStatusIndicator is a component used for important notifications that stretch across the entire width of the screen to draw the user's attention. Buttons rendered as children of this component will be colored to contrast appropriately with the gradient background.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the styles:

    ```js
    @import '@console/pal/Components/ResourceStatusIndicator/styles';
    ```

3.  Import the component:

    ```js
    import { ResourceStatusIndicator } from "@console/pal/Components";
    ```

4.  Render the component.

For general component usage review the [Components Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/components/overview).
