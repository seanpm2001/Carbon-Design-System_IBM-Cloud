## Description
ProgressBar is a component used for displaying progress, either as a series of stages or a percentage toward completion.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the styles:

    ```js
    @import '@console/pal/Components/ProgressBar/styles';
    ```

3.  Import the component:

    ```js
    import { ProgressBar } from '@console/pal/Components';
    ```

4.  Render the component.

For general component usage review the [Components Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/components/overview).
