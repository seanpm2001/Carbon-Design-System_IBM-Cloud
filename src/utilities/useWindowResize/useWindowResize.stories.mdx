# useWindowResize

## Description

A common hook for reacting to window resize events. The only input is the listener function that will get passed the resize event object. This hook can be used in multiple places and will result in only a single event listener being added to the window object, which will call each listener function passed to this hook. The listeners are debounced for 250 milliseconds before being called.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the utility

    ```js
    import { useWindowResize } from 'carbon-for-ibm-cloud/utilities';
    ```

3.  Use the utility

    ```js
    useWindowResize(input);
    ```

## Example

<Source language="json" code={`function MyComponent() {
