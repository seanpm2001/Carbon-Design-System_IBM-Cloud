## Description

The CardLayout Layout component is meant for putting Cards onto a page in
different arrangements. It uses carbon's grid system, and is tested with pal's
Card component, but it can be used with any other elements.

The PageHeader is the standard
[Cloud PAL PageHeader](https://pages.github.ibm.com/ibmcloud/pal/components/page-header/code).

## Usage

1.  Install the
    [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use)
    library
2.  Import the styles:

    ```js
    @import '@console/pal/Components/CardLayout/styles';
    ```

3.  Import the component:

    ```js
    import { CardLayout } from "@console/pal/Layouts";
    ```

4.  Render the component.

For general Layouts usage review the
[Layout Docs](https://pages.github.ibm.com/ibmcloud/pal/development-guidelines/layouts/overview).
